# Growth Diagnosis Examples

**[English](README.md)｜[简体中文](README.zh.md)**

All company details and metrics in these examples are simulated; the diagnostic mechanisms are realistic.

| Example | Diagnosis path | Learning goal |
| --- | --- | --- |
| [SaaS activation](saas-activation.md) | [Activation diagnosis](../README.md#activation-diagnosis) | Separate onboarding completion from first value, compare alternative constraints, and choose a bounded activation experiment. |
| [Consumer retention](consumer-retention.md) | [Retention diagnosis](../README.md#retention-diagnosis) | Match the retention interval to the usage cycle, read cohorts without hiding contradictory evidence, and prioritize restored value over reminders. |
| [SEO acquisition pages](seo-acquisition.md) | [Acquisition diagnosis](../README.md#acquisition-diagnosis) | Distinguish traffic growth from qualified retained demand, test the limiting SEO mechanism, and select the appropriate execution route. |
