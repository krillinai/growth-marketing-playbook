# 推荐与增长循环

> 推荐与增长循环把产品价值或用户活动转化成更多分发、内容、数据或价值。循环必须闭合并产生持续质量，而不只是邀请。

**[English](README.md)｜[简体中文](README.zh.md)**

## 核心问题

- 哪项有价值产出会创造下一项输入？
- 用户为什么愿意分享或邀请？
- 循环创造留存增量用户，还是只创造补贴活动？

## 跨公司模式

有效模式通常把产品价值、目标行为、衡量周期和经济或质量护栏连接起来。不同公司使用的表面策略不同，但都需要验证早期信号能否延伸到持续价值。

## 公司证据

| 公司或人物 | 证据与方法 |
| --- | --- |
| Facebook / Meta | 好友连接、邀请、网络价值，以及口碑与病毒传播的区别。 |
| Duolingo | 可分享进度、可识别品牌、习惯与学习身份。 |
| 字节跳动 | 裂变激励、K 因子、产品互导与反作弊。 |
| Lovable | 产品演示、员工、客户、发布与自然分发。 |

## 关键指标

- 邀请或分享率
- 接收者转化与留存质量
- 经过留存调整的 K 因子
- 循环周期与完成率
- 增量、成本、作弊与蚕食

## 可以迁移的方法

1. 画出从价值事件到下一次价值事件的完整循环。
2. 定义参与者动机与接收者价值。
3. 衡量持续增量质量，而不只看发送量。
4. 增加作弊、疲劳、平台与信任护栏。

## 不应该直接复制的部分

- 不要强迫分享或索取不必要权限。
- 产品价值或反作弊能力薄弱时，不要复制裂变奖励。

## 证据与局限

- 公司证据主要来自参与者访谈和公开材料，不是完整内部因果评估。
- 公司规模、市场、产品频率、监管与数据能力限制直接迁移。
- 发布方公布的增长与收入数字不用于证明单项方法有效。
