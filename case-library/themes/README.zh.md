## 持续更新规则

1. 先把公司专属证据加入公司页。
2. 按增长环节与横向能力给证据分类。
3. 只更新真正相关的主题对照页。
4. 只有新证据实质性支持或挑战现有模式时，才修改跨公司结论。
5. 保留来源日期、证据类型、参与者视角和适用边界。
6. 把发布方公布的增长、ARR、估值、客户数与 DAU 主张同因果结论分开。
