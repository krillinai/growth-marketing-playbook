# 激活

> 激活是新用户已经获得有意义产品价值的第一项证据。它是连接后续留存的行为假设，而不是完成一个引导界面。

**[English](README.md)｜[简体中文](README.zh.md)**

## 核心问题

- 哪种结果能证明用户获得了价值？
- 哪项早期行为可以预测持续使用？
- 不同意图的用户应该如何进入不同路径？

## 跨公司模式

有效模式通常把产品价值、目标行为、衡量周期和经济或质量护栏连接起来。不同公司使用的表面策略不同，但都需要验证早期信号能否延伸到持续价值。

## 公司证据

| 公司或人物 | 证据与方法 |
| --- | --- |
| Airtable | 分群引导、模板、第一个有用工作流与第四周协作。 |
| Facebook / Meta | 好友连接与 Magic Moment 作为早期价值代理。 |
| Sean Ellis | Must-have 价值、Aha Moment，以及激活作为留存桥梁。 |

## 关键指标

- 首次价值时间
- 按分群和来源划分的激活率
- 第一周与第四周留存激活
- 激活后的深度、广度或协作
- 错误、放弃与求助护栏

## 可以迁移的方法

1. 访谈留存用户并观察其首次成功路径。
2. 定义一项可能预测持续价值的可观察行为。
3. 只有产品路径真正变化时才进行分群。
4. 通过 Cohort 和护栏验证关系。

## 不应该直接复制的部分

- 不要复制通用 Aha Moment 或激活阈值。
- 不要把完成向导、关闭 Tooltip 或发出邀请当作已交付价值。

## 证据与局限

- 公司证据主要来自参与者访谈和公开材料，不是完整内部因果评估。
- 公司规模、市场、产品频率、监管与数据能力限制直接迁移。
- 发布方公布的增长与收入数字不用于证明单项方法有效。
